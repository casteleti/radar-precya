# RADAR PRECYA — FLUXO REVISADO (VERSÃO APROVADA)

## Filosofia do Produto

O Radar Precya não é um ERP.

O Radar Precya não é um sistema financeiro.

O Radar Precya é um motor de decisão para clínicas de estética.

Seu objetivo é responder:

> "Posso vender este procedimento por esse valor sem prejudicar minha margem?"

O foco é:

* simplicidade;
* clareza;
* confiança na decisão;
* precificação baseada em custo real;
* desconto com segurança;
* experiência mobile-first.
